# tmux-project-starter

A command line tool to start a new nvim session, server and all that is needed
for a great workflow with tmux

## Install

  Clone this repo `git clone https://github.com/Theacebutler/tmux-project-starter.git $HOME/.tmux-project-starter`

  Add this line to your bashrc file: `export PATH="$HOME/.tmux-project-starter/tmux-project-starter:$PATH"`

## Run

  `cd` in to the root directory of the project you want to start and 
  run `tmux-project-starter`

## Dependencies

  tmux
  nvim
  z (zoxide)
